# Project Handover Manual: Taskly API & Dashboard

## 1. Executive Summary
Taskly is a sophisticated Task Management solution designed for high-performance teams. It bridges the gap between a robust Spring Boot backend and a premium, high-fidelity frontend dashboard.

## 2. Technical Architecture

### üèó Backend Architecture
- **Framework**: Spring Boot 3.2.0
- **Database**: H2 (In-memory) for development; easy transition to PostgreSQL.
- **ORM**: Spring Data JPA with Hibernate.
- **Migration**: Flyway (Automatic schema management).
- **Communication**: RESTful API with JSON support.
- **Security**: Custom `ApiKeyAuthFilter` enforcing `X-API-KEY` verification for all `/api/**` routes.

### üé® Frontend Design Philosophy
- **Style**: Modern Glassmorphism.
- **Interactions**: Vanilla JavaScript with Lucide Icon integration.
- **Visuals**: HSL-based color palettes, CSS-based blurs, and micro-animations for a premium feel.

## 3. Core Functionalities

### User Management
- Create users with unique email validation.
- Paginated user list retrieval.
- Automated avatar generation via Dicebear API.

### Task Management
- **Lifecycle**: Create, Update, Patch Status, and Delete.
- **Filtering**: Multi-dimensional filtering by Priority (High/Medium/Low) and Status (Todo/Progress/Done).
- **Relational Integrity**: Tasks are linked to Users as "Assignees."

## 4. Operational Guide

### Deployment & Execution
The project is configured to run on **Port 8081** by default to prevent conflicts with standard Windows system services (like IIS or BranchCache) that often occupy 8080.

**Command to Run:**
```powershell
$env:JAVA_HOME = "C:\Program Files\Java\jdk-17.0.5"
..\maven\apache-maven-3.9.12\bin\mvn.cmd spring-boot:run
```

### Access Points
- **System Dashboard**: `http://localhost:8081/index.html`
- **Database Console**: `http://localhost:8081/h2-console`
- **Base API Path**: `http://localhost:8081/api`

## 5. Security & Authentication
The system uses a static Header-based authentication.
- **Header Key**: `X-API-KEY`
- **Header Value**: `my-secure-api-key-12345`
